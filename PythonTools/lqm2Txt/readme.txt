LG smarphones comes with its own Memo Application "QUICKMEMO" that stores files as ".jlqm" files

Users switching from LG to Other Brands find it difficult retrive ".jlqm".

Here comes the automated scripts to facilitate the users :)

steps:
1. Copy all your ".jlqm" files to a folder
2. Modify the "config.ini" file for extracting input_file path (Note: enter the folder path where you copied your files)
3. Run "lqm_to_text.py" file
4. Find the ".txt" file with the same name
